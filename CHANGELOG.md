# Changelog

All notable changes to Plenipo public documentation are documented here.

## [0.1.0-alpha] - 2026-06-14

### Added

- Production guides: deployment, DID hosting, agent onboarding, payments.
- API reference for relay, registry, sidecar, websocket, and error codes.
- Operations runbook, troubleshooting, backup/restore.
- Echo agent (`did:web:echo.plenipo.dev`) and `plenipo ping` onboarding docs.

### Updated

- Error reference documents `docs_url` field on Core and Registry responses.
- Operator webhook contract expanded beyond `balance.low`.
